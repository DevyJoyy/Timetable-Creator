This is a Timetable Creator Website 
I've created this to help make time management with modules a bit easier , it is 100% free , there is no login required .

Hosted safely via GitHub pages 
Try it out here : https://devyjoyy.github.io/Timetable-Creator/ 

What you can do with the Timetable Creator Website :
- create timetable : Quickly build a timetable from scratch.
- Save & Load (.xml) : You can save your data as an .xml file and load multiple files later, it automatically merges and creates your timetable for you!
- Print-Ready : Export your timetable as a PDF so you can print it .
  
( For now it works on desktops or Laptop [a device with a wide screen as the two listed ])
